# Outfit Checkers - VS Code-ready static site

## Included files
- index.html
- styles.css
- app.js
- kvkk.html
- acik-riza.html

## How to run
1. Open this folder in VS Code.
2. Install the Live Server extension if you want preview.
3. Open `index.html` and run with Live Server.
